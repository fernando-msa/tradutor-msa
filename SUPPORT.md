# Suporte

## Como pedir ajuda
- Abra uma issue em: https://github.com/fernando-msa/Tradutor-MSA-Extensao/issues
- Inclua navegador, versão da extensão e passos para reproduzir.

## Tipos de solicitação
- Bug report
- Solicitação de funcionalidade
- Dúvida de segurança/privacidade

## SLA comunitário
Projeto open source sem SLA comercial formal.
