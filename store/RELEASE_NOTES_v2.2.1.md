# Release Notes - Tradutor MSA v2.2.1

## Texto curto (Edge/Firefox)

Release de manutenção com correções de confiabilidade, validação automatizada e alinhamento do pacote de publicação.

## Texto completo (Edge/Firefox)

- Timeout seguro e tratamento de resposta inválida no fluxo de tradução.
- Recuperação de quota no storage local com limpeza do cache antes de falhar.
- Feedback visível quando a seleção do menu de contexto excede o limite.
- Melhorias de acessibilidade no popup e no estado ocupado do fluxo.
- Validação mínima automatizada e empacotamento alinhado entre local e workflow.

## Observacao de privacidade para listing

Ao traduzir, o texto e enviado para a MyMemory API (servico externo). A extensao nao possui backend proprio. Historico, cache e preferencias ficam salvos localmente no navegador.
