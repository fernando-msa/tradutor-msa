# Release Notes - Tradutor MSA v2.2.0

## Texto curto (Edge/Firefox)

Refatoracao completa do popup com melhorias de UX, acessibilidade e confiabilidade, incluindo ajustes de permissao/storage e maior transparencia de privacidade.

## Texto completo (Edge/Firefox)

- Refatoracao tecnica do fluxo principal de traducao no popup.
- Melhorias de UX e acessibilidade, com feedbacks de estado mais claros.
- Correcao de robustez no fluxo de traducao e historico.
- Reducao e endurecimento de permissoes no manifesto (MV3).
- Politica de privacidade e documentacao de seguranca revisadas.
- Melhor compatibilidade entre Edge e Firefox para operacoes locais e avaliacao.

## Observacao de privacidade para listing

Ao traduzir, o texto e enviado para a MyMemory API (servico externo). A extensao nao possui backend proprio. Historico, cache e preferencias ficam salvos localmente no navegador.
