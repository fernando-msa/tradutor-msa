# Código de Conduta

Este projeto adota comunicação respeitosa, objetiva e inclusiva.

## Esperado
- Respeito mútuo.
- Feedback técnico construtivo.
- Colaboração transparente.

## Não permitido
- Assédio, discriminação ou ataques pessoais.
- Divulgação maliciosa de dados sensíveis.

## Moderação
Casos graves podem resultar em bloqueio de participação no repositório.
